# bridgeServer
The bridge server is used to create and submit transactions to the CELO network
# Documentation
https://documenter.getpostman.com/view/3143535/U16nKirS
# To run
1.clone this repo
2.pip3 install -r requirements.txt
3.python3 run.py

The bridge server will run on port 8002 be default. You can change the port by changing the value inside the run.py
